API Reference

Base URL: https://screenshotly1.p.rapidapi.com — authenticate with your X-RapidAPI-Key header.

MethodEndpointDescription
POST /v1/screenshots Sync capture — returns result inline, or a job ID if still processing after 30s
POST /v1/jobs Async capture — enqueues immediately and returns a job ID for polling
GET /v1/jobs/{id} Poll a job by ID; returns status and a presigned download URL when complete
GET /v1/jobs List your jobs with pagination (page, pageSize)
GET /v1/account View your plan tier, requests used this month, and account details

Request Parameters

url string

Public URL to capture (HTTP/HTTPS). Either url or html is required.

html string

Raw HTML string to render. Alternative to url.

format enum

png · jpeg · webp · pdf. Default: png.

width / height integer

Viewport size in pixels. Range: 100–3840 × 100–2160. Default: 1280 × 800.

fullPage boolean

Capture the full scrollable page, not just the visible viewport. Default: false.

waitForSelector string

CSS selector to wait for before the capture is taken.

waitMs integer

Additional wait time in ms after page load (0–60,000).

webhookUrl string

POST the result here when complete. Paid plans only.

quality integer

JPEG/WebP quality 1–100. Ignored for PNG and PDF. Default: 90.

pdfPageSize string

PDF page size: A4, Letter, Legal, etc. PDF format only.
